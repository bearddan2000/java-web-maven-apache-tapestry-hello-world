# java-web-maven-apache-tapestry-hello-world

## Description
A POC for apache tapestry framework rendering html page.

## Tech stack
- jetty

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
[Available at](http://localhost/example/index)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
